Class: TreeNode

TreeNode

Class the describes a simple node in a treew data structure. Each filter is modeled as a element in a tree and this is how the dependencies are modeled and evaluated, dependencies are represented as a parent child relationship. Here we provide helper methods for evalutaing these relationships.

new TreeNode()

initialise the tree node with no parent and no children
