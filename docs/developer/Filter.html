_buildAPIUrl(url)

When the config.data function returns a string this is used to make a network call. You can also use format strings with parent filter values here

Parameters:
Name Type Description
url String

url from the config.data function
Returns:

the url with any format strings evaluated

Type
String
Example

Using format parameters in a data url

// The {{SomeFilter}} value will be replaced by the current value of that filter
// The filter used must be a parent of this filter
{
 dependsOn: [ 'SomeFilter' ]
 data: () => 'http://www.myapi.com/endpoint/{{someFilter}}/data'
}
